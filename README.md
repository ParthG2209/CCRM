# Campus Course & Records Manager (CCRM)

A simple Java console application for managing students, courses, enrollments, and grades.

## Requirements

- JDK 17+
- Maven 3+

## Build and run

```bash
mvn clean package
java -jar target/ccrm-1.0-SNAPSHOT.jar
```

On first run, use the menu shown in the console to add students, courses, offerings, enrollments and grades.
